The Journey

아무것도 모르는 상태에서 시작했습니다

Started With Zero Knowledge

코딩을 배운 적도, 트레이딩을 해본 적도 없었어요. 그냥 시작했고, 수십 번 실패했고, 그게 지금의 시스템이 됐습니다.

No coding background. No trading experience. Just started, failed dozens of times, and kept going. That became this system.

v1 — 첫 알림이 울리던 날 시작
v1 — The Day the First Alert Fired Start
제대로 동작하는지도 몰랐지만, 텔레그램에 메시지가 왔을 때의 그 기분은 아직도 기억해요.
Didn't even know if it worked properly. But when the Telegram message came through — that feeling is still with me.
6,899건 분석해서 내 봇을 직접 껐습니다 결정
Analyzed 6,899 Cases. Then Shut Down My Own Bot. Decision
뉴스가 매매에 도움이 될 것 같아 만들었는데, 데이터를 보니 해킹 뉴스만 플러스였어요. 데이터가 말하면 따라가는 수밖에 없었습니다.
Built a news bot thinking it would help trading. The data showed only hack-related news had positive impact. When the data talks, you follow.
567건 분석하고 배포하지 않기로 했습니다 보류
Analyzed 567 Cases. Decided Not to Deploy. Held
손절이 73%였어요. 만든 기능을 안 쓰는 것도 결정이라는 걸 그때 배웠습니다.
Stop-loss rate was 73%. Learned that choosing not to use what you built is also a decision.
8,271건 데이터 분석 — 고승률 필터 발견 발견
8,271 Data Points — High Win-Rate Filter Discovered Discovery
데이터가 쌓이면서 패턴이 보이기 시작했어요. 특정 조건에서만 작동하는 필터를 발견했습니다.
As data accumulated, patterns emerged. Found a filter that only works under specific conditions — but consistently does.
구글 시트가 10M 셀을 넘겼습니다 위기
Google Sheets Hit the 10M Cell Limit Crisis
사후추적 열이 정상 40개에서 949개로 폭발했어요. 버그를 찾아서 수동 복구하고, 재발 방지 로직을 3번 갈아치웠습니다. v43, v44, v45 전부 이 문제 때문에 나왔습니다.
Tracking columns exploded from the normal 40 to 949. Tracked down the bug, recovered manually, rewrote the prevention logic three times. v43, v44, v45 all shipped because of this one issue.
지금 — v76, 6개 봇 동시 운영 중 현재
Now — v76, Running 6 Bots Simultaneously Present
업비트 알림봇, 업비트 M4, 바이낸스, 유튜브봇, 일반뉴스봇, 청산모니터. 맥미니 한 대가 24시간 돌아가고 있습니다.
Upbit alert, Upbit M4, Binance, YouTube, General News, and Liquidation Monitor bots. One Mac Mini running 24/7.
